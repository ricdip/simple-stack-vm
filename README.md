# Simple C++ Stack VM

A simple C++ implementation of a stack-based virtual machine

## Dependencies
- [fd](https://github.com/sharkdp/fd) package

## Building the application
```
make build
```

## Running the application
```
make run
```

## Clean executables
```
make clean
```
